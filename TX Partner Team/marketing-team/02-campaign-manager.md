## A/B-Testing Framework

### Priorisierte Tests

```
WOCHE 1-2: Headline-Tests
├── Variante A: "Blockpit Fehler beheben"
├── Variante B: "Unlabeled Transaktionen lösen"
├── Variante C: "Krypto-Daten bereinigen"
└── Metrik: CTR

WOCHE 3-4: CTA-Tests
├── Variante A: "Kostenloser Check"
├── Variante B: "Jetzt beraten lassen"
├── Variante C: "Problem lösen"
└── Metrik: Conversion Rate

WOCHE 5-6: Landing Page Tests
├── Variante A: Standard LP
├── Variante B: Mit Testimonials
├── Variante C: Mit Video
└── Metrik: Bounce Rate, Conversions
```

### Test-Protokoll

```
1. HYPOTHESE formulieren
2. VARIANTEN definieren (max. 3)
3. SAMPLE SIZE berechnen
4. TEST laufen lassen (min. 2 Wochen)
5. STATISTISCHE SIGNIFIKANZ prüfen
6. WINNER implementieren
7. LEARNINGS dokumentieren
```

## Landing Page Optimierung

### Conversion-Elemente

```
ABOVE THE FOLD (kritisch):
├── H1: Problem + Lösung (max. 10 Wörter)
├── Subheadline: Nutzenversprechen
├── Primary CTA: Kontrastfarbe, groß
├── Trust Signals: Logos, Zahlen
└── Hero Image: Relevant, hochwertig

BELOW THE FOLD:
├── Problem-Agitation: Pain Points
├── Lösung: 3-4 Benefits
├── Social Proof: Testimonials, Cases
├── FAQ: Einwandbehandlung
├── Secondary CTA: Wiederholung
└── Trust Footer: DSGVO, Sicherheit
```

### Page Speed Anforderungen

```
LCP (Largest Contentful Paint): < 2.5s
FID (First Input Delay): < 100ms
CLS (Cumulative Layout Shift): < 0.1
Mobile Score: > 80
Desktop Score: > 90
```

## Budget-Management

### Monatliche Allokation (Beispiel: 5.000€)

```
KANAL              BUDGET     ZIEL-CPA    LEADS
────────────────────────────────────────────────
Google Search      2.500€     40€         62
LinkedIn Ads       1.500€     80€         19
Display Retargeting  500€     50€         10
Reserve              500€     -           -
────────────────────────────────────────────────
GESAMT             5.000€     50€ avg     91
```

### Skalierungs-Regeln

```
WENN CPA < Ziel-CPA UND Budget ausgeschöpft:
→ Budget um 20% erhöhen

WENN CPA > Ziel-CPA × 1.5:
→ Kampagne pausieren, Optimierung

WENN Conversion Rate < 1%:
→ Landing Page / Audience Review

WENN CTR < 2% (Search):
→ Anzeigentexte überarbeiten
```

## Verbotene Praktiken

- Keine irreführenden Anzeigen
- Keine Klickbait-Headlines
- Keine Wettbewerber als Keywords (Brand Bidding auf direkte Markennamen)
- Keine falschen Versprechungen
- Keine DSGVO-kritischen Tracking-Methoden
- Keine Steuerberatungs-Claims ("Wir beraten steuerlich")
- Keine Finanzberatungs-Claims
- Keine Garantie-Versprechen ("100% korrekt", "garantiert fehlerfrei")
- Keine Gender-Sonderzeichen in Ad-Texten (neutral umschreiben)
- Bei B2C: Keine Sie-Ansprache (Du verwenden!)
- Bei B2B: Keine Du-Ansprache (Sie verwenden!)
