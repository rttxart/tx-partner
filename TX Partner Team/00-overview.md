## Wann welchen Agenten nutzen?

### Frontend-Entwickler
- Neue Features implementieren
- Bestehende Komponenten erweitern
- JavaScript-Funktionalität hinzufügen
- Tailwind CSS anpassen
- Responsive Anpassungen

### UI/UX Designer
- Neue Layouts entwerfen
- Design-System erweitern
- Farben, Typografie, Spacing anpassen
- Animationen konzipieren
- Mobile-First Designs

### SEO/GEO Spezialist
- Meta-Tags optimieren
- Schema.org Markup erweitern
- Interne Verlinkung verbessern
- Content für KI-Zitierbarkeit optimieren
- Keyword-Integration

### Search Console Experte
- Indexierungsprobleme lösen
- Crawl-Fehler beheben
- Core Web Vitals verbessern
- Sitemap-Probleme lösen
- Mobile Usability Issues

### Performance Engineer
- Ladezeiten optimieren
- Assets komprimieren
- Caching konfigurieren
- Critical CSS extrahieren
- Lazy Loading implementieren

### QA & Testing
- Cross-Browser Testing
- Accessibility prüfen
- HTML/CSS validieren
- Funktionalität testen
- Regressionstests

## Workflow: Feature-Implementierung

```
1. UI/UX Designer    → Design & Layout planen
2. Frontend-Entwickler → Code implementieren
3. SEO/GEO Spezialist → SEO-Optimierung
4. Performance Engineer → Optimierung
5. QA & Testing       → Qualitätssicherung
```

Alle Agenten haben Zugriff auf:

## Nutzung

1. Öffne den gewünschten Agenten
2. Kopiere den System-Prompt in Claude/ChatGPT
3. Beschreibe dein Feature/Problem
4. Der Agent liefert produktionsreifen Code/Lösung

---

## Qualitätsstandards

Jeder Agent hält sich an:

- **Keine Breaking Changes** ohne Warnung
- **Mobile-First** Responsive Design
- **Accessibility** WCAG 2.1 AA
- **Performance** Core Web Vitals optimiert
- **DSGVO-konform** Keine externen Ressourcen
- **Dokumentation** Code kommentiert

## Schnellstart-Guide

### Neues Feature implementieren

```
1. UI/UX Designer    → "Entwirf eine Testimonial-Sektion"
2. Frontend-Entwickler → "Implementiere diese Testimonial-Sektion"
3. SEO/GEO Spezialist → "Optimiere die Sektion für SEO"
4. Performance Engineer → "Prüfe die Performance"
5. QA & Testing       → "Teste auf allen Geräten"
```

### SEO-Problem lösen

```
1. Search Console Experte → "Analysiere warum /seite nicht indexiert wird"
2. SEO/GEO Spezialist    → "Optimiere Meta-Tags und Schema"
3. QA & Testing          → "Validiere die Änderungen"
```

### Performance verbessern

```
1. Performance Engineer → "Analysiere LCP von 4s"
2. Frontend-Entwickler  → "Implementiere Critical CSS"
3. QA & Testing        → "Teste auf allen Geräten"
```

### Design-Änderung

```
1. UI/UX Designer      → "Redesigne den Hero-Bereich"
2. Frontend-Entwickler → "Setze das neue Design um"
3. QA & Testing       → "Responsive Testing"
```

## Agent-Prompts (Copy & Paste)

### Für Claude/ChatGPT

1. Öffne die gewünschte Agent-Datei
2. Kopiere den gesamten Inhalt
3. Füge ihn als System-Prompt ein
4. Stelle deine Frage/Aufgabe

### Beispiel-Prompts

**Frontend-Entwickler:**
> "Erstelle eine FAQ-Accordion-Komponente mit Animation"

**UI/UX Designer:**
> "Entwirf ein Pricing-Grid mit 3 Paketen"

**SEO/GEO Spezialist:**
> "Optimiere /leistungen für 'krypto dokumentation'"

**Search Console Experte:**
> "5 Seiten zeigen 'Erkannt – zurzeit nicht indexiert'. Was tun?"

**Performance Engineer:**
> "LCP ist 3.8s auf Mobile. Wie verbessern?"

**QA & Testing:**
> "Führe einen Accessibility-Audit für /kontakt durch"
